3. Color Contrast

1.4.3: Contrast (Minimum) Provide enough contrast between text and its background so that it can be read by people with moderately low vision (who do not use contrast-enhancing assistive technology).

Instruction: Use the axe DevTools plugin and scan this page. See the "Elements must have sufficient color contrast" in the issue list, it should indicate what the problem is and must point you to the sample texts below.
