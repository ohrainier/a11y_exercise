2. Images

1.1.1: Non-text Content Make information conveyed by non-text content accessible through the use of a text alternative.

Instruction: Use the axe DevTools plugin and scan this page. See the "Images must have alternate text" in the issue list, it should indicate what the problem is and must point you to the image below.

Test Area

Problem: An image may not directly convey information that can be understand easily. Especially if the user is relying at assistive technologies like a Screen Reader.

Recommendation: Ensure that all images in the page are equipped with an "alt" attribute and a meaningful text value in order for the Screen Reader to understand what the image conveys.
